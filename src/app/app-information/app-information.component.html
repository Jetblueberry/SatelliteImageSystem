About us

VIETNAM SATELLITE IMAGES SYSTEM (VSIS) is a satellite images website providing the necessary tools for users to work with different types of satellite image data within the territory of Vietnam.

The project team chose the topic of building a system that be able to store and manage satellite image data on the territory of Vietnam, which will serve many research purposes having practical value in the fields of meteorology and geology in Vietnam such as researching rainfall, warning of flood locations, measuring temperature of provinces and cities,... like other satellite imaging systems in the world

Understand us

VIETNAM SATELLITE IMAGES SYSTEM (VSIS) is available open software built on the Angular Framework platform, a popular web framework in system construction. Leaflet map library is used to support integration of map types to display satellite images on them.

User can access to code of this system on this link github . If any questions, please send to email of developer: phananhle2002@gmail.com

Core Platform Software Used

  • Leaflet:

    Providing all types of embedding map.

  • Geoserver:

    Supporting WMS service to query satellite images from virtual machine server.

  • Angular Framework:

    Web Framework that used to develop this software.
